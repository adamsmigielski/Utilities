# Utilities
Utility libraries for C++.

Build system
CMake - The following libraries are added:
 - basic,
 - containers,
 - helpers,
 - math,
 - memory,
 - task.
It requires following definitions:
 - UTILITIES_COMPILER*
 - UTILITIES_OS*
 - NDEBUG
Global settings should be set before Utilities is added.

Requirements
Depends on:
 - Platform.hpp file,
 - math depends on xmmintrin.h.
